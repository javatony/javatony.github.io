MY WORK

Park.ly

Connecting people and parking.

Park.ly is a web application similar to Airbnb but for parking. It connects people looking for parking spots with people willing to offer their private space for rent.

Built using PostgreSQL, Express, Angular and Node. Geocoding and map rendering is provided by Google Maps and Mapbox API while Twilio, is used for text alerts for reservations and cancellations.

Citysights

Let the local attractions FIND you!

Using the geolocation, CitySights displays local attractions that are within walking distance.

Built using Ruby on Rails, Javascript, JQuery and Bootstrap. Utlized HTML5's built in geolocation and Gooogle Maps API to render the map and pin drops.
